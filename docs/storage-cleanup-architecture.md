# 存储清理安全架构

此功能当前仅有架构，不执行扫描或删除。

## 阶段

1. 只读扫描：统计路径、大小、文件年龄、占用状态和来源。
2. 风险分类：区分可重建缓存、临时文件、日志、用户数据和系统数据。
3. 预览清单：逐项展示将影响的路径与预计释放空间。
4. 用户确认：按类别确认，不提供“无提示全部删除”。
5. 工具原生命令：优先使用 `npm cache`、`pip cache`、`cargo cache` 等受支持命令。
6. 审计与恢复：保存操作清单；可移动到隔离区的内容优先隔离而非永久删除。

## 永久保护

- Windows、Program Files、用户文档和桌面。
- Git 项目、工作区和未知源码目录。
- DevEnv Manager 受管运行时与当前指针。
- 浏览器 Cookie、密码、登录状态和配置。
- 正在使用、被锁定或来源不明的文件。

## 非目标

- 不做注册表清理。
- 不做驱动清理。
- 不绕过 Windows 权限。
- 不静默删除或创建自动清理计划任务。
